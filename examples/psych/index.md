# 心理治疗理论范畴论分析

## 范畴定义

| 疗法 | 范畴 $\mathcal{C}$ | 对象 | 态射 |
|------|------------------|------|------|
| CBT | $\mathbf{CBT}$ | $\{X, B, N, E\}$ | 认知链路 |
| 心理动力学 | $\mathbf{Psycho}$ | $\{X, R, N, U, E\}$ | 移情纤维 |
| 人本主义 | $\mathbf{Human}$ | $\{X, S, R, E\}$ | 实现同构 |

## 核心结构

- **CBT**：$X \xrightarrow{f} B \xrightarrow{g} N \xrightarrow{h} E$，线性因果链
- **心理动力学**：$X \to R \right arrows N \to E$，纤维结构
- **人本主义**：$X \xrightarrow{f} R \xrightarrow{g} S \xrightarrow{h} E$，同构映射
